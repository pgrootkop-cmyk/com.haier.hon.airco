# Changelog

All notable changes to this project will be documented in this file.

## [0.0.1] - 2025-01-01

### Added
- Initial release
- hOn API client with OAuth2 authentication (MFA/SSO supported)
- Air conditioner driver with cloud pairing flow
- HVAC modes: Auto, Cool, Heat, Dry, Fan Only, Anti-Freeze
- Fan speed control: Auto, Low, Medium, High
- Swing mode: Off, Vertical, Horizontal, Both
- Eco Pilot: Off, Avoid Me, Follow Me
- Toggle switches: Silent, Rapid, Sleep, Eco, Health, Screen Display, Beep
- Indoor and outdoor temperature sensors
- Configurable API poll interval
- Flow cards: triggers, conditions, and actions for all features
- English and Dutch translations
